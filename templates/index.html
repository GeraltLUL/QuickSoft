Наш стек

Узнайте, c чем мы работаем

Unity

Unity – кроссплатформенная среда разработки игр. Вся наша игра работает на этом движке. Многопользовательская состовляющяя была реализована при помощи специальной библиотеки «Netcode».

Flask Framework

Flask – очень распространенный Python фреймворк для создания веб-приложений. Бекенд этого сайта работает именно на нём.

Python

Python – наше главное средство для анализа игровых данных. С его помощью можно легко обрабатывать большие массивы данных.

MongoDB

MongoDB – Система управления базами данных, работающая на NoSQL-модели. Данные о наших пользователях и их игровых сессиях мы храним с помощью этой системы.

Наш прогресс

Познакомьтесь с историей разработки проекта

  • Январь 2022

    Формирование команды

    После выступления на конференции ИТСАУ–2021 с материалом, который позже станет главной "фичей" творческого проекта – системы анализа, было принято решение о поиске команды для реалзиации многопользовательской игры. Мы понимали, что нам нужны заинтересованные люди, которые не боятся писать код, поэтому мы придумали несколько задач по программированию и опубликовали открытый контест на платформе codeforces, участие в котором приняли более 10 студентов, по результатам которого мы отобрали в команду лучших.

  • Апрель 2022

    Работа в процессе…

    К апрелю у нас была большая часть нужных механик, некоторые из которых впоследствии были успешно переписаны в связи с повышением уровня знаний команды. Параллельно большинство участников команды посещает курс ДПО по разработке компьютерных игр при институте.

  • Август-сентябрь 2022

    Последние приготовления перед защитой

    К сентябрю команда заканчивает работу над многопользовательской состовляющей проекта, системой аналитики и этим сайтом. Также мы готовимся к выступлению на фестивале студенческих проектов "Войти в IT".
